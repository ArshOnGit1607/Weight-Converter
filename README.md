# Weight Converter

## Overview
This project is a simple and interactive web application that allows users to convert weights between different units (such as kilograms, pounds, grams, etc.). The app provides a user-friendly interface for quick and accurate conversions.

## Features
- Convert weights between multiple units
- Responsive and modern UI
- Instant conversion results

## Tech Stack
- **HTML**: Structure of the web application
- **CSS**: Styling and layout for a clean, responsive design
- **JavaScript**: Handles conversion logic and user interactions

## How It Works
1. Enter a weight value and select the unit you want to convert from.
2. Choose the unit you want to convert to.
3. The app instantly displays the converted value.

## Usage
Open `index.html` in your browser to use the Weight Converter.

## Project Structure
- `index.html`: Main HTML file
- `style.css`: Stylesheet for the app
- `app.js`: JavaScript logic for conversions
